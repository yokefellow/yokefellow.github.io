## 开发者工具 <Badge text="new" type="tip"/>

### 1.搜索引擎

::: tip
搜索引擎的优先级推荐：Google + 英文 > Bing（国际版） + 英文 >> 百度。常见的搜索引擎指令可以参考[如何用好谷歌等搜索引擎（知乎）](https://www.zhihu.com/question/20161362/answer/14180620)。相关搜索技巧可以参考前阿里 Java 资深研发工程师（目前就职于 Gradle）、开源届大牛张博（花名：彰德）分享的[程序员搜索技巧：使用搜索解决 95%的技术问题（哔哩哔哩）](https://www.bilibili.com/video/BV1Nh411Z7wH/)
:::

### 2.技术社区

::: tip
首推 Stack Overflow。如果在搜索引擎中使用准确的英文关键词来描述问题，搜索结果通常会显示 Stack Overflow 中以存在的话题。如何在技术社区中提出一些有意义的问题？可以参考 Eric S. Raymond, Rick Moen 十多年前编写的[How To Ask Questions The Smart Way](http://www.catb.org/~esr/faqs/smart-questions.html)，中文版：[提问的智慧](https://github.com/ryanhanwu/How-To-Ask-Questions-The-Smart-Way/blob/master/README-zh_CN.md)
:::
